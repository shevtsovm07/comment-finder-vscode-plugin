## Описание

Comment Finder - это расширение для Visual Studio Code, которое автоматически находит и подсвечивает комментарии с ключевыми словами TODO и FIXME в вашем коде. Помогает не забывать о важных задачах и необходимых исправлениях.

## Возможности

- **Поиск комментариев** - находит TODO и FIXME в однострочных и многострочных комментариях
- **Визуальная подсветка** - выделяет найденные комментарии желтым цветом
- **Уведомления** - показывает количество найденных комментариев
- **Быстрый запуск** - работает через палитру команд VS Code
